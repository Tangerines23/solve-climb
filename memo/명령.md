✅ 해결 방향
1. 퀴즈 카드와 결과 카드 구조 통일
퀴즈 카드 구조 예시
tsx
<div className="quiz-card">
  <div className="question">4 + 9 = ?</div>
  <input className="answer-input" />
  <button className="submit-button">제출</button>
</div>
결과 카드 구조 예시
tsx
<div className="quiz-card result-mode">
  <div className="result-title">시간 종료!</div>
  <div className="result-score-label">최종 점수:</div>
  <div className="result-score">{score}</div>
  <button className="submit-button result-confirm-button">확인</button>
</div>
quiz-card를 그대로 쓰고, 결과 모드일 때 result-mode 클래스를 추가

버튼 클래스는 동일하게 submit-button을 유지하면서 스타일만 분기 처리

2. CSS에서 버튼 위치 통일
css
.quiz-card {
  display: flex;
  flex-direction: column;
  justify-content: flex-start;
  height: 300px;
  padding: 24px 16px;
}

.submit-button {
  margin-top: auto;
  align-self: center;
}
margin-top: auto로 버튼을 카드 하단으로 밀어냄

align-self: center로 가운데 정렬

퀴즈 카드와 결과 카드 모두 동일한 버튼 위치를 갖게 됨

3. 점수 영역은 가운데 정렬 유지
css
.result-score {
  font-size: 2rem;
  color: #3182f6;
  text-align: center;
  margin-bottom: 16px;
}
