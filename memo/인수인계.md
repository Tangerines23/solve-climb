# === 프롬프트 시작 ===

[프로젝트 개요]
- 프로젝트명: Solve Climb
- 목적: 다양한 지식 카테고리를 빠르게 풀어나가는 스피드 퀴즈 앱
- 모드: 타임 어택(Time Attack), 서바이벌(Survival)
- 카테고리: 수학, 언어, 논리, 상식 등
- 구조: 레벨 기반 성장 시스템 (클리어 시 다음 레벨 해금)

[현재 구현 상태]
1. ThemeProvider 적용 및 다크 모드 설정 완료
2. MathQuizPage.tsx에서 문제 출제, 정답 입력, 제출 처리 구현됨
3. 피드백 시스템:
   - 정답 시: 초록색 깜빡임 + 점수 pop 애니메이션 + 문제 페이드 전환
   - 오답 시: 빨간색 흔들림 + 진동 + 문제 페이드 전환
4. 타이머 기능: Timer.tsx 생성, 타임 어택 모드에서 30초 카운트다운
5. 모드 선택: ModeSelector.tsx 생성, 카드형 UI로 타임 어택/서바이벌 선택
6. 레벨 시스템: LevelUnlock.tsx 생성, 클리어 시 다음 레벨 잠금 해제 UI 구현
7. 스타일: MathQuizPage.css에 토스 스타일 기반 카드형 UI, 빠른 애니메이션 적용

[UI 요소 참고 (토스 퀴즈 스타일 기반)]
- 배경: #121212 (다크 모드)
- 카드: #ffffff, 둥근 모서리, 그림자
- 버튼: 넓고 둥글게, 색상은 토스 블루(#3182f6), 오답용 레드(#f04452)
- 피드백: 깜빡임, 흔들림, 진동 모두 0.1~0.15초 이내
- 점수: 증가 시 pop 애니메이션
- 문제 전환: 페이드 아웃 → 페이드 인
- 타이머: 상단 중앙에 초시계 표시
- 폰트: 굵고 간결하게, 여백은 넉넉하게

[개선 필요 사항]

- 색상 및 버튼 스타일을 토스 디자인 시스템에 맞게 리디자인
